# How To Run

1. Clone the repo 
```bash
$ git clone https://github.com/zhang-yixiang/20241206.git
```
2. Change to the repo dir
```bash
$ cd pdf-ai-client
```
3. Install dependencies
```bash
$ npm install
```
4. Run the dev env.
```bash
npm run dev
```
5.Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
